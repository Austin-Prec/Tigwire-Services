# Tigwire Services

Marketing site and private writing area for Tigwire Services, a
professional cleaning company serving homes, offices, schools, hotels, and
banks across Malawi.

Built with React, TypeScript, Vite, Tailwind CSS, and Supabase.

**First time setting this up?** See [SETUP.md](./SETUP.md) for the full,
step-by-step guide to connecting a new Supabase project, running the
database migrations, and deploying the site.
